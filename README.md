# CLASSE A

App React com interface simples para leitura de historico e sugestao operacional em Bac Bo.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

O projeto ja esta preparado para deploy no Vercel com:

- framework `vite`
- build command `npm run build`
- output directory `dist`
- rewrite global para `index.html`

Para ativar deploy por commit:

1. Inicialize ou use este repositorio Git.
2. Suba para GitHub, GitLab ou Bitbucket.
3. Importe o repositorio no Vercel.
4. A cada push na branch configurada, o Vercel fara um novo deploy.
